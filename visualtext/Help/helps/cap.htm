cap

Purpose

Pre action that succeeds if leaf token has capitalized first letter.

Syntax

<fromRuleEltNumber,toRuleEltNumber>cap();

fromRuleEltNumber - type: int
toRuleEltNumber - type: int

Returns

Remarks

Example

@PRE
<1,1>cap();
@RULES
_noun <- _xALPHA@@
