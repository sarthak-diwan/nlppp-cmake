prlit

Purpose

OBSOLETE.  Print the literal string string to fileName.

Syntax

prlit(fileName, string)

fileName - type: str
string - type: str

Returns

Remarks

Old-style.  The output file (fileName) must be set up in advance using openfile, fileout or an output statement.
