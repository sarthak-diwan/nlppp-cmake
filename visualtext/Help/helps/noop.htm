noop

Purpose

Perform no POST Action. This is a convenience to disable the default single() reduce action.

Syntax

noop()

Returns

Remarks

Example

# Increment the count of nouns,
# without reducing _noun to anything.
@CHECK
++G("noun count");
@POST
 noop();  # If absent, then _noun will reduce to _xNIL.
@RULES
_xNIL <- _noun @@

See Also

single, POST Actions
