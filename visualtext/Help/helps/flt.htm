flt

Purpose

Convert string or num to float type, if possible.

Syntax

returnedNumber = flt(string)
returnedNumber = flt(num)

returnedNumber - type: int
string - type: str
num - type: int

Returns

Remarks

This function also accepts a float arg.

Example

@CODE
G("val") = flt("345.67");
@@CODE
