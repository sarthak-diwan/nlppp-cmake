merge

Purpose

Perform a single tier reduction that dissolves each top level node in the matched phrase.

Syntax

merge()

Returns

Remarks

Example

# Remove the two _adjs nodes
# from the parse tree, merging
# their children under a new
# _adjs node.
@POST
merge();
@RULES
_adjs <- _adjs _adjs @@
