Opening the Phrase Tool

The Phrase Tool allows you to view the phrase of a concept in the Knowledge Base.  You can delete, add or edit nodes and their attributes using the Phrase Tool.  Concepts in the Knowledge Base with phrases are indicated by a yellow icon in the Knowledge Base hierarchy.  

To view the phrase for a concept in the Knowledge Base:

  1. In the KB Editor, navigate to the concept in the hierarchy with a yellow icon.  

  2. Right-click on the icon and select Phrase Tool.  The Phrase Tool opens up in the Workspace.  

To delete a node from a phrase:

  1. Select the node in the list of nodes displayed in the Phrase Tool.

  2. Click the Delete button.  

  3. At the confirmation prompt, answer Yes.

To add a node to a phrase:

  1. In the Phrase Tool, click the Add button.

  2. In the dialog box, enter a name for the node and click OK. The new node is added to the phrase.

To add attributes to a node:

  1. Select the node in the list of nodes displayed in the Phrase Tool.

  2. Click on the Attributes button.  The Attribute Editor is launched.

  3. To add an attribute to node, click Add Attribute.  In the Name column, provide a name and hit the return key.  At this time you can provide a value for the name.  

To change the order of the nodes in the phrase tool:

  1. Select the node in the list of nodes.

  2. Use the blue arrow buttons to move the node to the desired location.
