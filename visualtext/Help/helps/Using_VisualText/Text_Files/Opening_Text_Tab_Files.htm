Opening Text Tab Files

To open a file in the Text Tab:

  • In the Text Tab, double-click on the file.  The file opens in the Workspace.  You can also use the right-click menu in the Text Tab and select Edit/View to open the file.

Tip: If the file icon in the Text Tab is already highlighted, you can also click on the View Current Text button on the Workspace Toolbar to open the file.
