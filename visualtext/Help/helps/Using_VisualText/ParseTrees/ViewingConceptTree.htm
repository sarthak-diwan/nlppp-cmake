Viewing Concept Trees

The concept tree shows you which nodes were built for a specific pass in the analyzer sequence.  

To view nodes created in a parse tree:

  1. In the Ana Tab, right-click on a pass.

  2. Select View > Concept Tree.

Note: To view concept trees, you must run the analyzer with the Generate Logs button on.

 

Tip: You can also view a concept tree by selecting the View Concept Tree button on the Debug Toolbar.
