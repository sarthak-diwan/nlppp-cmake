varstrs

Purpose

Create an empty multi-string-valued global variable with name varname.

Syntax

varstrs(varname)

varname - type: str

Returns

Remarks

Obsolete, slated for removal.  Use NLP++ array code instead, e.g.,

G("tmp") = 0;
G("tmp") = "abc";
G("tmp")[1] = "def";

Example

@CODE
varstrs("tmp");  # Create a global variable named "tmp".
